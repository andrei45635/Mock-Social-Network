<p> Mock Social Network built using JavaFX, JDBC and Gradle. </p>
<p> Current functionalities: </p>
<ul>
  <li> Login </li>
  <li> Register </li>
  <li> View friends list </li>
  <li> Add friend </li>
  <li> Remove friend </li>
  <li> Send friend request </li>
  <li> Accept/Reject friend request </li>
  <li> Send message </li>
</ul>
